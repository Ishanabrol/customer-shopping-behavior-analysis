# Customer Shopping Behaviour Analysis

## Project Overview
This project analyzes customer shopping behaviour using retail transaction data to uncover purchasing patterns, customer segments, and revenue-driving factors. The goal is to transform raw transactional data into actionable business insights that support data-driven decision-making in retail environments.

The project demonstrates an end-to-end **Data Analytics workflow**, combining data preprocessing, database analytics, and business intelligence visualization.

---

## Objectives
- Analyze customer spending patterns and purchase frequency
- Compare behaviour across demographic groups
- Evaluate the impact of subscriptions and discounts
- Identify top-performing product categories
- Perform customer segmentation (New, Returning, Loyal)
- Build interactive dashboards for business insights
- Provide strategic business recommendations

---

## Dataset Information
- **Total Records:** 3,900 transactions
- **Features:** 18 attributes
- **Key Variables:**
  - Age
  - Gender
  - Product Category
  - Purchase Amount
  - Subscription Status
  - Discount Applied
  - Shipping Type
  - Review Rating
  - Purchase Frequency
